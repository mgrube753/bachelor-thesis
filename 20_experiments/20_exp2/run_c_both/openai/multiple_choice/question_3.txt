Frage: Untersuchen Sie folgende Situation: Ein schneller Sender überträgt Datenrahmen deutlich schneller, als ein langsamer Empfänger sie verarbeiten kann. Welches Verfahren sollten Sie auf der Sicherungsschicht (Data Link Layer) anwenden, um das Problem gemäß ISO-OSI-Modell zu lösen?

Antwortmöglichkeiten:
- ~Ein Flusskontrollverfahren einsetzen, das dem Sender signalisiert, wann der Empfänger weitere Rahmen annehmen kann
- Ein Routing-Protokoll konfigurieren, das alternative Pfade zum Empfänger auswählt
- Die physikalische Sendeleistung erhöhen, um die Bitfehlerrate zu verringern
- Eine Ende-zu-Ende-Verschlüsselung der Nutzdaten in der Sitzungsschicht aktivieren
