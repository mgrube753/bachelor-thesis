Frage: Detaillieren Sie anhand des bereitgestellten Textes, welche Kernaufgaben die Data Link Layer (Sicherungsschicht) des ISO-OSI-Modells übernimmt und welche zwei spezifischen Herausforderungen sie dabei adressiert, um eine effektive Datenübertragung zu gewährleisten.

Antwort: ~Die Data Link Layer hat die Hauptaufgabe, eine rohe Übertragungseinrichtung in eine Leitung zu verwandeln, die frei von unentdeckten Übertragungsfehlern erscheint, indem sie die eigentlichen Fehler vor der Network Layer verbirgt. Dies wird erreicht, indem der Sender Eingabedaten in Datenframes zerlegt und diese sequenziell überträgt; bei zuverlässigem Dienst bestätigt der Empfänger den korrekten Empfang jedes Frames durch ein Bestätigungsframe. Eine spezifische Herausforderung, die die Data Link Layer adressiert, ist die Flusskontrolle, um zu verhindern, dass ein schneller Sender einen langsamen Empfänger mit Daten überflutet. Eine weitere Herausforderung, die insbesondere in Broadcast-Netzwerken auftritt, ist die Kontrolle des Zugriffs auf den gemeinsam genutzten Kanal, welche durch eine spezielle Unterschicht, die Medium Access Control (MAC) Sublayer, gelöst wird.
