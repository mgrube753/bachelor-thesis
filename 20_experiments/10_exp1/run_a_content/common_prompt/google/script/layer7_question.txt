KORREKTE ANTWORT:
Der Application Layer (Schicht 7) ist stark abhängig von der jeweiligen Anwendung, wodurch wenig Systematisierung möglich ist.

FRAGE:
Welche der folgenden Aussagen beschreibt ein wesentliches Merkmal des Application Layers (Schicht 7) im ISO-OSI-Modell, wie im gegebenen Text dargestellt?

ANTWORTMÖGLICHKEITEN:
A) Der Application Layer (Schicht 7) ist stark abhängig von der jeweiligen Anwendung, wodurch wenig Systematisierung möglich ist.
B) Die Protokollelemente des Application Layers lassen sich aufgrund ihrer Struktur sehr gut systematisieren.
C) Eine der Kernaufgaben des Application Layers ist die Festlegung der physischen Übertragungselemente.
D) Das Verhalten der Anwendung auf Protokollelemente ist in diesem Layer fest vorgegeben und nicht anpassbar.
