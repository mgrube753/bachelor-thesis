Frage: Das im Text geschilderte Szenario, in dem ein Browser mittels HTTP eine Webseite von einem Server anfordert, illustriert vor allem welches Grundprinzip des ISO/OSI-Schichtenmodells?

Antwortmöglichkeiten:
- ~Das Prinzip, dass Interaktionen für den Benutzer auf der Anwendungsschicht abstrahiert werden, während die Details der darunterliegenden Schichten verborgen bleiben.
- Die Notwendigkeit, vor jeder Datenübertragung zunächst die Transportverbindung auf der Transportschicht durch einen Three-Way-Handshake aufzubauen.
- Den Mechanismus, mit dem die Sicherungsschicht Kollisionen im gemeinsamen Medium auflöst.
- Die Nutzung von Routingtabellen der Vermittlungsschicht zur Pfadfindung zwischen autonomen Systemen.
